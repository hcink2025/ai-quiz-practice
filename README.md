# AI易错题练习系统

一个简洁美观的在线答题练习系统，包含60道AI人工智能相关易错题。

## 功能特点

- 📚 **60道精选题目** - AI运维、机器学习、数据分析等知识点
- 🎯 **两种模式** - 练习模式（即时反馈）和考试模式（提交后显示结果）
- 📊 **实时统计** - 进度、正确率实时更新
- 💡 **即时答案** - 练习模式中选错立即显示正确答案
- 📱 **响应式设计** - 支持手机、平板、电脑

## 使用方法

直接打开 `index.html` 即可使用，无需安装任何依赖。

## 知识点分布

| 类别 | 题数 |
|-----|------|
| AI系统运维 | ~15道 |
| 机器学习基础 | ~10道 |
| 图像处理/计算机视觉 | ~6道 |
| 数据分析方法 | ~8道 |
| AI应用领域 | ~8道 |
| 项目/产品设计 | ~5道 |
| 特征工程/模型评估 | ~5道 |

## 技术栈

- 纯HTML + CSS + JavaScript
- 单文件应用，无需服务器
- 数据内置，开箱即用
